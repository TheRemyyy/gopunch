# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-29

### Added
- **Initial Release**
- **Multi-Protocol Support**: HTTP, TCP, DNS, and SSL checks.
- **Continuous Monitoring**: `watch` mode for real-time uptime tracking.
- **Alerting System**: Discord and Slack webhook integration with cooldown.
- **Concurrency Control**: Semaphore-based parallel execution.
- **Flexible Output**: Support for Table, JSON, CSV, and Minimal formats.
- **Resilience**: Automatic retries with exponential backoff.
- **Documentation**: Comprehensive guides for architecture, configuration, and development.
